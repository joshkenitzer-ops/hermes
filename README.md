# ⚠️ Deprecated

This repository is archived. Development continues at [`iris`](../../iris).

**Hermes** was renamed **Iris** on 2026-07-21, following a naming collision with commercial "Hermes AI" agent products. The rename kept the messenger-goddess theme consistent with the rest of the Lore tool suite.

**Architecture also changed.** This repository holds the original application: six deterministic services, a fixed pipeline, calling a model at set points. Iris is being rebuilt as an agentic harness: a specification functioning as pinned context, a set of deterministic tools the model must invoke, and a model operating under both. Not a refactor of what is here. A different design.

This history is kept for reference. The commit log, the v0.1 through v0.9 spec line, and the session handoffs document real decisions and real failure modes that shaped the current spec, including a documented gap between decisions made in conversation and decisions written down, which the current architecture is built to close.

No further work happens here.
